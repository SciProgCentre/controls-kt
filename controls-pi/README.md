# Module controls-pi

Utils to work with controls-kt on Raspberry pi

## Usage

## Artifact:

The Maven coordinates of this project are `space.kscience:controls-pi:0.3.0`.

**Gradle Kotlin DSL:**
```kotlin
repositories {
    maven("https://repo.kotlin.link")
    mavenCentral()
}

dependencies {
    implementation("space.kscience:controls-pi:0.3.0")
}
```
